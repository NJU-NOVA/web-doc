## React

React 是一个非常优秀且流行的 MVVM 框架，使用它你将会因为声明式的编程方式有完全不一样的开发体验，你可能会觉得很神奇并且开始好奇这是怎么做到的。

但是目前，你应该先学习基本使用也不迟，[React 官方文档](https://zh-hans.reactjs.org/docs/hello-world.html)是你入门最好的选择，目前阶段，你只需要学习`核心概念`即可。
